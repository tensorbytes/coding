#### 个人总结

##### 52. N皇后 II
```
. 该解题思路主要是 

1.使用bit位来记录位置(三个整形)
2.位运算的与或后 取反 可放置皇后的位置( bit位为1), 
3.  循环 利用 n & -n 取到 最低位的1 放置皇后,  再利用 n & (n-1)消去 最低位的1 
    从而代表已经尝试了该种组合


```
